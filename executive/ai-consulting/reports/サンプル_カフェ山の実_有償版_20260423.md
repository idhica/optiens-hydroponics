## 2. 自動化提案（7件）

以下の提案はすべて**エージェントAI（自律型AI）**による自動化です。チャットAIに質問してコピペするような使い方は一切提案しません。AIエージェントが既存のデータソース（写真・口コミ・仕入れデータ等）を自動で読み取り、API経由で各サービスに直接アクションを実行します。

### 2-1. Instagram投稿の完全自動化（優先度: A）

- **現状**: 料理の写真は撮っているが、キャプション作成に時間がかかり週1回以下
- **自動化の仕組み**:
  1. オーナーが料理写真をGoogleフォトに保存（普段通り写真を撮るだけ）
  2. **AIエージェントが毎日11:00に自動起動**
  3. Googleフォト APIで新しい写真を自動検出 → 画像認識で料理名・食材を推定 → キャプション＋ハッシュタグを自動生成
  4. Instagram Graph API経由で**自動投稿**（人の操作なし）
- **オーナーの作業**: 写真を撮る（普段通り）。それ以降は一切不要
- **期待効果**: 投稿頻度 週1→毎日。投稿作成時間 月2.7h→**0h（完全自動）**

### 2-2. Googleマップ口コミの自動巡回・返信（優先度: A）

- **現状**: 月10-15件の口コミに対し、返信率50%以下
- **自動化の仕組み**:
  1. **AIエージェントが週2回（水曜・金曜9:00）自動起動**
  2. Google Business Profile APIで新着口コミを自動取得
  3. 過去の返信履歴を参照し、御社のトーンで返信文を自動生成
  4. LINEでオーナーに返信ドラフトを通知
  5. **承認ボタンを押すだけ**で、APIを通じてGoogleに自動投稿
- **オーナーの作業**: 通知を見て承認する（1件30秒）
- **期待効果**: 返信作業 月2.5h→**月0.3h（承認のみ）**。返信率90%以上でMEO順位改善

### 2-3. LINE公式メニュー配信の完全自動化（優先度: A）

- **現状**: 開設済みだが休止中。友だち数120人
- **自動化の仕組み**:
  1. 仕入れデータ（発注書メール or スプレッドシート）をAIエージェントが自動読み取り
  2. 入荷食材＋過去メニュー＋季節情報から、本日のおすすめ配信文を**自動生成**
  3. LINE Messaging API経由で毎日14:30に**友だちへ自動配信**
  4. 人の操作は一切不要。データソースを整備するだけ
- **オーナーの作業**: なし（仕入れデータが読める形式であればAIが自律的に処理）
- **期待効果**: 配信作業 月3h→**0h（完全自動）**。リピーター来店頻度 +10%

### 2-4. メニュー企画の自動提案（優先度: B）

- **現状**: 月2回のメニュー替え。食材の組み合わせ案出しに2時間/回
- **自動化の仕組み**:
  1. 仕入れ食材リストをGoogleスプレッドシートで管理（既存の発注データを流用）
  2. **AIエージェントが月2回（1日・15日の朝）自動起動** → スプレッドシートを自動読み取り
  3. 入荷食材＋過去メニュー＋季節＋原価データを分析し、5-10案を自動生成
  4. LINEでオーナーに提案が届く → 選ぶだけ
- **オーナーの作業**: 提案を見て選ぶだけ（15分）
- **期待効果**: 企画時間 月4h→**月0.5h**

### 2-5. 予約管理のデジタル化（優先度: B）

- **現状**: 紙ノートで管理。電話取り逃がし月5-8件
- **自動化の仕組み**:
  1. Googleカレンダーで予約管理に移行（無料）
  2. LINE公式の予約機能を有効化 → 顧客が24時間セルフ予約可能に
  3. **AIエージェントが毎晩20:00に自動起動** → 翌日の予約をカレンダーから読み取り → LINEで自動リマインド
- **オーナーの作業**: 電話予約をカレンダーに登録するだけ（LINE予約は自動入力）
- **期待効果**: 予約管理時間 月5h→月1.5h（**月3.5時間削減**）。取り逃がしゼロ化

### 2-6. 帳簿入力の自動化（優先度: B）

- **現状**: 月末にレシートを手入力。月4時間
- **自動化の仕組み**:
  1. freee会計（月額¥1,980）に移行
  2. 銀行口座・クレジットカードをAPI連携 → 取引データが**自動取込**・仕訳候補を自動生成
  3. レシートはスマホで撮影 → OCR自動読取 → 仕訳に自動反映
- **オーナーの作業**: レシート撮影のみ。銀行・カード経由の取引は完全自動
- **期待効果**: 帳簿入力 月4h→月0.5h（**月3.5時間削減**）

### 2-7. スタッフ連絡の自動化（優先度: C）

- **現状**: LINEグループで連絡。シフト変更・業務連絡が混在
- **自動化の仕組み**:
  1. シフト表をGoogleスプレッドシートで管理
  2. **AIエージェントが毎晩19:00に自動起動** → スプレッドシートから翌日のシフトを読み取り → 該当スタッフにLINEで自動通知
  3. 業務連絡もスプレッドシートの「連絡事項」シートに記入 → AIが適切なタイミングで自動配信
- **オーナーの作業**: スプレッドシートに記入するだけ（既存の紙ノートの代替）
- **期待効果**: 連絡確認時間 月5h→月2h（**月3時間削減**）

## 3. 自動化アーキテクチャ

本提案の全体像です。**AIエージェントがデータソースを自動で読み取り、API経由で各サービスにアクションを実行**します。オーナーが新しいツールを覚えたり、AIに質問したりする必要はありません。

```
┌──────────────────────────────────────────────┐
│  データソース（AIエージェントが自動読み取り）  │
│                                              │
│  Googleフォト ── 料理写真                    │
│  スプレッドシート ── 仕入れ・シフト・メニュー │
│  Googleカレンダー ── 予約データ              │
│  Google Business Profile ── 口コミ           │
│  freee API ── 銀行・カード取引               │
└──────────────┬───────────────────────────────┘
               │ 自動読み取り
    ┌──────────▼──────────┐
    │   AIエージェント     │
    │  （自律型・スケジュール実行）│
    │                      │
    │ ┌──────────────────┐ │
    │ │ 毎日 11:00       │ │ → Instagram自動投稿
    │ │ 毎日 14:30       │ │ → LINE配信自動送信
    │ │ 水・金 9:00      │ │ → 口コミ巡回→返信ドラフト通知
    │ │ 1日・15日 8:00   │ │ → メニュー提案自動生成
    │ │ 毎晩 19:00       │ │ → シフトリマインド
    │ │ 毎晩 20:00       │ │ → 翌日予約リマインド
    │ └──────────────────┘ │
    └──────────┬───────────┘
               │ API経由で自動実行
    ┌──────────▼──────────────────────┐
    │   外部サービス（自動アクション） │
    │                                │
    │  Instagram Graph API（投稿）   │
    │  LINE Messaging API（配信）    │
    │  Google Business Profile（返信）│
    │  freee API（帳簿）             │
    └────────────────────────────────┘

    オーナーの操作:
    ・写真を撮る（普段通り）
    ・口コミ返信を承認する（通知が来たら）
    ・メニュー提案を選ぶ（通知が来たら）
    → それ以外はすべてAIエージェントが自動実行
```

### なぜ「AIエージェント」なのか

- **チャットAI**: 人間が質問文を入力 → 回答を読む → コピペする。人間が介在する
- **AIエージェント**: データを自動読み取り → 判断 → API経由で実行。**人間の介在なし**

本提案はすべてAIエージェント方式です。オーナーが新しいツールの画面を開いて操作する必要はありません。

このアーキテクチャが、以下の導入支援の工数と費用の根拠になります。

### 導入で重要なポイント: データソースの整備

AIエージェントが自律的に動くためには、**AIが読める形式のデータ**が必要です。導入支援では以下のデータ整備も行います。

| 現状 | 整備後 | AIエージェントの動作 |
|---|---|---|

| 仕入れはFAXで発注 | スプレッドシートに転記 or メール転送設定 | 食材データを自動読み取り → メニュー提案・配信 |

**既存の業務をなるべく変えずに、AIが読めるようにデータの置き場所を整理する**のが導入支援の核心です。
