# AI支援事業 — 用語マスタ

## 目的
業務フロー定義・無料版vs有償版_定義・サイトページ・契約書テンプレ・営業資料における用語のゆれを排除する。
すべての成果物作成時に本マスタを参照する。新しい表現を導入する場合は本マスタを更新してから他文書に反映する。

## 運用ルール
- 文書作成・編集時は必ず本マスタを参照
- 用語のゆれを発見したら本マスタの「正」表現に統一
- 新規用語の導入は本マスタへの追記が前提（事後追加禁止）
- バージョンアップ時は改訂履歴を残す

### 技術固有名詞の使い分け

**社外発信（サイト・営業資料・無料版レポート）では使用しない:**
- Supabase / Astro / Claude Code / Anthropic API / Next.js / GitHub Actions / Vercel など

**有料レポート・提案書・契約書では使用OK:**
- 顧客が技術選定の判断材料として必要な範囲で記載

**社内文書では使用OK:**
- 業務フロー定義・契約書テンプレ・社内マスタなど

理由: 案件ごとに最適な技術を選定するため、サイトに固定表現を置くと将来の選択肢を狭める。

## トーン・ブランド関連

### 採用しないトーン
- 煽り口調：「圧倒的」「革命」「破壊」「逆転」「神アプデ」「衝撃」
- 数値煽り：「10倍」「20倍」「業界NO.1」「爆発的成長」
- FOMO訴求：「使わないと致命的」「乗り遅れる」「今すぐ」
- 対立フレーム：「都会を席巻」「同業を打倒」

### 採用するトーン
- 円相哲学（調和・最適化・循環）と整合
- 誠実・落ち着き・実証ベース
- 顧客メリット中心（売り込みではなく価値提案）
- 「弊社」を主語に
- 競合は「業界では〜」「一般的には〜」とフラットに
