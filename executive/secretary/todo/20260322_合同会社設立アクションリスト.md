# 合同会社設立 アクションリスト

> 作成日: 2026-03-22
> ステータス: 進行中
> 目標: 2026年4月中に合同会社を設立し、7月からSF契約を法人に切替

---

## Phase A: 設立準備（3月下旬〜4月上旬）

### A-1. 設立事項の確定
- [ ] **会社名を決定**（「合同会社Optiens」or 別案）
- [ ] **本店所在地を決定**（自宅住所 or バーチャルオフィス）
- [ ] **資本金を決定** → 推奨: 10〜50万円（1,000万円未満で消費税2期免税）
- [ ] **決算月を決定** → 推奨: 3月（4月設立なら免税期間最大化・補助金スケジュールと整合）
- [ ] **役員報酬を決定** → 月50万円（初年度。期中変更不可）
- [ ] **事業目的を確定**（下記案を参照）

#### 事業目的（案）
```
1. 農業用設備及び水耕栽培システムの企画、開発、製造、販売及び保守
2. 農業IoTに関するソフトウェア及びSaaSの企画、開発、運営
3. データ分析、AI及びIoT技術を活用したコンサルティング
4. 情報処理システムの設計、開発、運用及び保守に関する業務
5. インターネットを利用した各種情報提供サービス
6. 前各号に附帯又は関連する一切の事業
```
※4番でSalesforce業務委託をカバー

### A-2. freee会社設立で書類作成（無料）
- [ ] [freee会社設立](https://www.freee.co.jp/launch/) にアカウント作成
- [ ] 上記の情報を入力 → 定款・登記書類を自動生成
- [ ] 電子定款の場合: freee経由で行政書士に依頼（5,000円、freee法人プラン契約で無料）
- [ ] 代表社員の印鑑を作成（ネット注文で2,000〜5,000円、届くまで数日）

### A-3. 印鑑証明の取得
- [ ] 個人の印鑑証明書を取得（市役所、300円）

---

## Phase B: 法人登記（4月中旬）

### B-1. 法務局に登記申請
- [ ] 必要書類一式を法務局に提出（甲府地方法務局 韮崎出張所）
- [ ] 登録免許税: 60,000円（収入印紙）
- [ ] 登記完了まで約1〜2週間

### B-2. 登記完了後の手続き
- [ ] 登記事項証明書（履歴事項全部証明書）を取得（600円）
- [ ] 法人の印鑑カードを取得

---

## Phase C: 設立直後の届出（4月下旬〜5月上旬）

### C-1. 税務関係
- [ ] **税務署**: 法人設立届出書（設立から2ヶ月以内）
- [ ] **税務署**: 青色申告の承認申請書（設立から3ヶ月以内）
- [ ] **税務署**: 給与支払事務所等の開設届出書
- [ ] **税務署**: 源泉所得税の納期の特例の承認申請書（従業員10人未満 → 半年に1回の納付に）
- [ ] **県税事務所**: 法人設立届出書（山梨県）
- [ ] **市役所**: 法人設立届出書（北杜市）

### C-2. 社会保険
- [ ] **年金事務所**: 健康保険・厚生年金保険 新規適用届
- [ ] **年金事務所**: 被保険者資格取得届（自分の分）
- [ ] 役員報酬月額50万円で届出 → 標準報酬月額の決定
- [ ] **「社保の窓口」を解約**（合同会社の社保加入が完了してから）

### C-3. 法人口座開設
- [ ] **第1候補: GMOあおぞらネット銀行**（審査が比較的通りやすい）
- [ ] **第2候補: 住信SBIネット銀行**
- [ ] **第3候補: PayPay銀行**
- [ ] 申込に必要: 登記事項証明書・代表者の本人確認書類・事業内容の説明
- [ ] ※メガバンクは避ける（KSC情報が2028年頃まで残る可能性）

### C-4. 会計
- [ ] **freee法人プランを契約**（ミニマム: 年26,136円）
- [ ] 法人口座をfreeeに連携
- [ ] 役員報酬の仕訳設定

---

## Phase D: 個人事業の整理（5〜6月）

### D-1. Salesforceクライアントとの調整
- [ ] 法人契約への切替を正式に相談
- [ ] 7月の契約更新タイミングで切替を目指す
- [ ] 請求書の宛先・振込先を法人に変更

### D-2. 個人事業の廃業
- [ ] **税務署**: 個人事業の廃業届（7月の法人切替と同時期）
- [ ] **税務署**: 青色申告の取りやめ届出書
- [ ] **税務署**: 給与支払事務所等の廃止届出書（該当する場合）
- [ ] 個人事業の最終確定申告の準備（1〜6月分）
- [ ] freee個人プランの解約（年間契約の満了タイミングを確認）

---

## Phase D-3. インボイス対応（9〜10月）

> ★重要: 設立直後にインボイス登録しないこと（4〜9月の免税期間を活かす）

- [ ] **9月**: インボイス登録申請（10月適用開始に間に合わせる）
- [ ] **9月**: 簡易課税制度選択届出書を提出（第5種・みなし仕入率50%）
- [ ] **10月〜**: インボイス発行開始（経過措置80%→50%縮小に対応）
- [ ] 消費税納税額の見込み: 年間約33万円（簡易課税適用時）
- [ ] SFクライアントにインボイス番号を通知

---

## Phase E: Optiens事業の本格始動（5月〜）

### E-1. 補助金リサーチ・申請準備
- [ ] 山梨県・北杜市の補助金公募を確認（年度初めに公募開始が多い）
- [ ] 小規模事業者持続化補助金の申請検討（上限50〜200万円）
- [ ] ものづくり補助金の公募スケジュール確認
- [ ] 認定支援機関（税理士・商工会）への相談

### E-2. 税理士の選定
- [ ] freee認定アドバイザーで山梨県対応の税理士を検索
- [ ] 初年度はスポット（設立届出 + 決算）で依頼を検討（15〜25万円）
- [ ] 北杜市商工会への相談（無料の経営相談あり）

### E-3. MVP開発（並行して進行）
- [ ] Docker Desktop インストール → Supabase ローカル開発環境
- [ ] GitHub に `optiens-iot` リポジトリ作成・push
- [ ] RPiの既存Pythonコードを `edge/` に移植
- [ ] Supabase CLI セットアップ
