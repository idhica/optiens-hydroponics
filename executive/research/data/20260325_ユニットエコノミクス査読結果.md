# 査読結果: ユニットエコノミクス算出方法 / 事業計画書 v2.0

**査読日**: 2026年3月25日
**総合評価**: **要修正**（数値の整合性に重大な矛盾あり）
**承認推奨**: No（修正後に再査読）

## 1. 数値の整合性 — 重大な問題あり

### 問題 1-1: 1株あたりコストの試算表と個別計算式の著しい乖離（重大）

**ユニットエコノミクスレポート 1-1節**の試算表では「1株あたり合計: 約18-45円」としているが、**同レポート内の個別計算式（1-2, 1-3, 1-4節）**から算出すると大幅に異なる。

| コスト項目 | 試算表の値（1-1節） | 個別計算式からの値 | 乖離 |
|---|---|---|---|
| LED電気代 | 8-20円 | **33.6円**（1-2節の計算結果） | 試算表の上限を13.6円超過 |
| 養液代 | 2-4円 | **6円**（1-3節の計算結果） | 試算表の上限を2円超過 |
| 減価償却費 | 2-5円 | **26円**（1-4節の計算結果） | 試算表の上限を21円超過 |
| 種子代 | 1-3円 | 1.5円 | 整合 |
| 水代 | 0.5-1円 | 記載なし（試算表通り） | — |
| 人件費 | 3-10円 | 記載なし（試算表通り） | — |
| 消耗品 | 1-2円 | 2円 | 整合 |
| **合計** | **18-45円** | **約73-78円** | **合計が約1.7-4倍の乖離** |

**原因の推定**: 試算表（1-1節）は「業界標準値ベースの理想的な範囲」で記載し、個別計算式（1-2〜1-4節）は「Phase 1の実際の設備構成（50W LED、20株、6年償却）」で計算しているため、前提条件が異なる。しかし試算表のヘッダーには「Phase 1試算値」と記載されており、両者はPhase 1の数値として一致すべき。

**修正案**:
- 試算表（1-1節）の数値を個別計算式の結果に合わせて更新する
- または、試算表は「業界標準の範囲」、個別計算式は「Optiens Phase 1の具体試算」と明確に区別するラベルを付与する
- エグゼクティブサマリーの「1株あたり生産コスト約15-40円」も修正が必要

### 問題 1-2: 損益分岐点計算の前提値と個別計算値の矛盾（重大）

**ユニットエコノミクスレポート 5-2節**のPhase 1損益分岐点計算:
```
月間固定費: 約2,500円
1株あたり変動費: 約8円
```

しかし、1-2節の計算ではLED電気代だけで33.6円/株。LED電気代は株数に比例しない固定費的性格が強いが、その場合「月間固定費2,500円」にLED電力コストが含まれているべき。

**LED月間電力コストの検算**:
```
0.05kW x 16h x 30日 x 30円/kWh = 720円/月（LEDのみ）
```

月間固定費2,500円にはLED+ポンプ+RPi+減価償却が含まれるとあるが、減価償却費だけで:
```
40,575円 / 6年 / 12ヶ月 = 約564円/月
```

LED 720円 + 減価償却 564円 = 1,284円で、ポンプ+RPi電力を含めると2,500円は概ね妥当な数字にはなるが、**変動費8円の内訳が不明瞭**。種子1.5円+養液?円+水0.5円+培地2円+包装?円で8円とするなら養液コスト（6円/株）との整合性が取れない。

**修正案**: 損益分岐点計算に使用した固定費・変動費の内訳を明示する表を追加する

### 問題 1-3: 事業計画書のPhase 2月間コストの不一致（中程度）

| 記載箇所 | Phase 2月間コスト |
|---|---|
| 4.2節（収益モデル） | 約5.3万円 |
| 6.2節（費用構造） | 約53,000-86,000円 |

4.2節では「約5.3万円」と下限のみ記載されており、6.2節の上限86,000円との差が大きい。Phase 2の月間利益が「2.7-12.7万円」（4.2節）とされているが、コスト上限86,000円 x 月間売上下限8万円の場合は**赤字**になる。

**修正案**: 4.2節のコストを「約5.3-8.6万円」に統一し、月間利益の下限を再計算する。月間売上8万円 - コスト8.6万円 = **-0.6万円（赤字）**となるため、最低売上目標の引き上げ、またはコスト圧縮策の追記が必要。

### 問題 1-4: Phase 2損益分岐点の固定費と費用構造の不一致（中程度）

ユニットエコノミクスレポート5-3節: 「月間固定費: 約80,000円（家賃+パート人件費+電気代+その他）」
事業計画書6.2節: Phase 2月間コスト合計は「約53,000-86,000円」

損益分岐点計算が上限値80,000円で計算されているのは保守的で良いが、事業計画書の下限値53,000円と一致しないことを補足すべき。

### 問題 1-5: Phase 1のコスト構造表の減価償却0円と計算式26円/株の矛盾（軽微）

事業計画書4.3節のコスト構造表では「減価償却（機器）: 0円（既存機器を使用）」としているが、ユニットエコノミクスレポート1-4節では再調達価格ベースで26円/株と計算している。

**判断**: 事業計画書は「キャッシュアウトベース」、ユニットエコノミクスレポートは「経済コストベース」と解釈すれば両立するが、読者（投資家・審査機関）に誤解を生む可能性がある。

**修正案**: 事業計画書の表に注記を追加: 「※既存機器のため追加キャッシュアウトはゼロ。経済コスト（再調達価格ベース）での減価償却は約564円/月（詳細はユニットエコノミクスレポート参照）」

## 7. その他の指摘

### 7-1: ファイル名とバージョンの不一致（軽微）

事業計画書のファイル名は `事業計画書_v2.0.md` だが、フロントマターの `version: 4.0` および本文の「第4.0版」と一致しない。

**修正案**: ファイル名を `事業計画書_v4.0.md` に変更するか、フロントマターを `v2.0` に修正する

### 7-2: バジルのASP表記（軽微）

事業計画書1.4節で「バジル（ASP $2,200、レタスより高マージン）」と記載。ASP $2,200は米国市場の数値であり、日本の事業計画書では円建ての数値（例: 卸売価格 2,461円/kg）を使用すべき。

**修正案**: 「バジル（卸売価格 約2,500円/kg、レタスの約3倍）」等に変更

### 7-3: ハイポニカ液肥の価格不一致（軽微）

ユニットエコノミクスレポート1-1節: 「ハイポニカ1L=約1,000円」
ユニットエコノミクスレポート1-3節: 「A液500mL + B液500mL = 約1,500円/セット」

1Lセットが1,000円なのか1,500円なのか不明。養液コストの計算結果に影響する。

**修正案**: 実際の購入価格を確認し統一する

### 7-4: 養液代の計算式の表記（軽微）

1-1節の養液代は「2-4円/株」、1-3節の計算結果は「6円/株」。1-3節の計算式自体は正しいが、1-1節の範囲に収まっていない。

## 結論

両文書とも構成・論理展開は良好で、CLAUDE.md禁止事項への違反はなく、試算値の明記も適切に行われている。しかし、**ユニットエコノミクスレポート内の試算表と個別計算式の間に約1.7-4倍の数値乖離**が存在し、これが事業計画書の損益分岐点・収益試算にも波及している。事業の根幹である「1株あたりコスト」の正確性が担保されていない状態であるため、**修正後の再査読を推奨**する。
