# IT導入支援事業者登録調査

## エグゼクティブサマリー

**結論: 合同会社Optiens（2026年4月6日設立）は、現時点でIT導入支援事業者への単独登録は不可能。**

最大の障壁は「法人税の納税証明書（直近1年分）」の提出要件。1期目の決算（2027年3月末）を迎えるまで、この書類を用意できない。コンソーシアム構成員としての参加も、法人としての納税証明書が必要なため同様に困難。

---

## 1. IT導入支援事業者の登録要件

### 1-1. 基本要件（法人の場合）

| 要件 | 内容 | Optiensの状況 |
|------|------|---------------|
| 法人格 | 日本国内で登録された法人 | OK |
| 納税証明書 | 税務署発行の直近1年分（その1又はその2） | NG: 1期目未了 |
| 履歴事項全部証明書 | 発行3ヶ月以内 | OK |
| GビズIDプライム | 法人代表者名義で取得 | 取得可能 |
| SECURITY ACTION宣言 | IPAの「一つ星」又は「二つ星」を宣言 | 無料で即日可能 |
| 販売実績一覧 | ITツールの販売実績（過去のもの） | 実績なしでも登録可能 |
| 反社会的勢力でないこと | 誓約書提出 | OK |

### 1-2. 設立からの期間制限

明示的な「設立N年以上」という要件は存在しない。しかし、**納税証明書（その1: 納税額等証明用 又は その2: 所得金額用）が実質的に1期以上の決算完了を要求**している。領収書・その3・その4は不可。電子納税証明書はPDF形式のみ有効。

### 1-3. 従業員数の要件

従業員数に関する明示的な下限要件はない。1名法人でも要件上は登録可能。ただし「補助事業者への十分な支援体制」が審査される。

### 1-4. IT導入実績

販売実績がなくても登録可能。ただし、実績があるほうが審査上有利。

### 1-5. セキュリティ要件

IPA「SECURITY ACTION」の宣言が必須。自己宣言制度であり、無料。Webから即日宣言可能。
- 一つ星: 「情報セキュリティ5か条」に取り組むことを宣言
- 二つ星: 「5分でできる！情報セキュリティ自社診断」を実施し基本方針を定め外部公開

出典: https://www.ipa.go.jp/security/security-action/it-hojo.html

### 1-6. その他の要件

- 補助金等の不正受給に関与していないこと
- 事業に必要な許認可を取得していること
- 労働関係法令を遵守していること
- 個人情報の取り扱いに関する適切な管理体制があること
- 事務局からの問い合わせに対応できる体制があること

## 2. 登録手続きの流れ

### 2-1. 手続きフロー

1. GビズIDプライムの取得（2週間程度）
2. SECURITY ACTION宣言（即日〜数日）
3. IT導入支援事業者ポータルサイトから登録申請
4. 必要書類のアップロード
5. 事務局による審査
6. 登録完了通知
7. ITツールの登録申請（別途）

### 2-2. 必要書類（法人）

- 履歴事項全部証明書（発行3ヶ月以内）
- 法人税の納税証明書（その1又はその2）★最大の障壁
- 販売実績一覧
- コンソーシアムの場合: コンソーシアム協定書

### 2-3. 審査期間

公式には明示されていないが、概ね2〜4週間程度。

### 2-4. 登録費用

無料。

出典: https://it-shien.smrj.go.jp/itvendor/flow/

## 3. ITツール登録

### 3-1. コンサルティングサービスの登録可否

**登録可能。** IT導入補助金2025から「活用コンサルティング」が補助対象に追加され、デジタル化・AI導入補助金2026でも引き続き対象。

### 3-2. カテゴリ分類

ITツールの大分類III「役務」に以下のカテゴリがある:

| カテゴリ | 内容 | Optiensサービスとの対応 |
|----------|------|-------------------------|
| カテゴリ5: 導入コンサルティング | ソフトウェア導入に際し、交付決定後〜導入開始までの費用 | AI活用診断レポート（15万円） |
| カテゴリ5: 活用コンサルティング | ITツール導入完了後6ヶ月間の利活用コンサルティング | AI顧問（月5万円） |
| カテゴリ6: 導入設定・マニュアル作成・導入研修 | 導入時の設定・研修費用 | AI研修（5万円） |
| カテゴリ7: 保守サポート | 運用保守費用 | 該当なし |

### 3-3. 重要な制約

- **カテゴリ5（コンサルティング）は1申請あたり1ツールのみ申請可能**
- **導入コンサルティング・活用コンサルティングは、カテゴリ1（ソフトウェア）の導入に紐づく必要がある**
- つまり、コンサルティング単体では補助対象にならない可能性が高い。何らかのソフトウェア（AIツール等）の導入が前提
- ソフトウェア（AIを含む）の定義が2026年度から明確化されており、AI機能を有するツールが推奨される

### 3-4. Optiensサービスの登録可能性

Optiensの「AI活用診断レポート」「AI顧問」をITツールとして登録するには、**何らかのソフトウェア/SaaS導入を組み合わせる必要がある**。例:
- Claude / ChatGPT等のAIツール導入 + 活用コンサルティング
- 業務用SaaS導入 + AI活用診断

コンサルティング単独での登録は要件を満たさない可能性が高い。

## 5. 1名法人・設立直後での実現可能性

### 5-1. 現状の評価

| 項目 | 評価 | 備考 |
|------|------|------|
| 1名法人での登録 | 制度上は可能 | 従業員数の下限要件なし |
| 設立直後の登録 | 不可能 | 納税証明書が提出不可 |
| コンソーシアム構成員 | 要検討 | 幹事社（既存IT企業）が必要 |
| 販売実績なし | 問題なし | 実績不要で登録可能 |

### 5-2. 設立直後の最大障壁

**法人税の納税証明書（その1又はその2）が提出できない。**

- 合同会社Optiensの第1期決算: 2027年3月末
- 決算後に確定申告 → 納税証明書取得: 2027年5月頃
- つまり、最短でも2027年6月頃まで登録不可

### 5-3. 代替手段

#### A. コンソーシアム構成員として参加
- 幹事社（1期以上の法人のみ）が必要
- 構成員にも納税証明書が求められるため、法人としての参加は困難
- 代表者が個人事業主としても開業届を出していた場合、個人の確定申告書・所得税納税証明書で構成員登録できる可能性あり

#### B. 2027年度からの登録を目指す
- 2026年度はIT導入支援事業者登録なしで事業を行う
- 2027年3月の1期目決算完了後、2027年度の登録に申請
- その間にAI活用診断の実績を積み、登録時の審査を有利にする

#### C. 既存のIT導入支援事業者と提携
- 既にIT導入支援事業者として登録済みの企業と業務提携
- Optiensのサービスをその事業者のITツールとして登録してもらう
- 手数料・マージンが発生するが、2026年度中に補助金スキームを利用可能

## 6. 総合判断と推奨アクション

### 短期（2026年度）

1. **IT導入支援事業者への単独登録は見送り**（納税証明書の壁）
2. **既存のIT導入支援事業者との提携を検討**（最も現実的）
   - 地域のIT企業やコンサル会社で既に登録済みの事業者を探す
   - Optiensのサービスをそのツールラインナップに追加してもらう形
3. **SECURITY ACTION宣言は今すぐ実施**（無料・即日、将来の登録に必要）
4. **GビズIDプライムは取得済みか確認**（未取得なら早期に申請）

### 中期（2027年度〜）

1. 1期目決算完了後（2027年5月頃）に納税証明書を取得
2. IT導入支援事業者として単独登録申請
3. 自社サービス（AI活用診断 + ソフトウェア導入パッケージ）をITツールとして登録
4. 顧客がデジタル化・AI導入補助金を活用してOptiensのサービスを利用可能に

### ITツール登録時の留意点

- コンサルティング単体では登録困難 → **ソフトウェア導入とセットのパッケージ化が必要**
- 例: 「AI業務効率化パッケージ」= AIツール（SaaS）導入 + 活用コンサルティング + 導入研修
- これにより通常枠（補助率1/2、最大150万円未満）の対象となり得る
