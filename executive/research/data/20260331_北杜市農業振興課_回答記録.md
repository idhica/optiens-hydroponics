# 北杜市農業振興課 回答記録（2026-03-31）

## 問い合わせ: 2026-03-27（メール）
## 回答: 2026-03-31（メール）

---

## 1. チャレンジ農業助成金 → **対象外**

- 水耕栽培事業は**助成金対象にならない**
- 理由: 「休耕田・遊休畑等を有効活用」することが前提。施設ではなく**農地の活用が条件**
- 法人（合同会社）での申請は可能だが、以下の条件あり:
  - 5年以上農業を継続して行う者
  - または農業経営改善計画の認定を受けた者
- 申請は随時受付、認定審査会は**年1回（9月頃）**
- 面談が必要

## 2. 認定新規就農者 → **対象外（兼業不可）**

- 兼業の場合は**申請できない**
- 対象: 「農業に専従するもの」で、年間従事時間等の要件を満たす必要
- **水耕栽培は農業として認定対象になる**（重要な確認事項）
- 認定手続き: 市と面談→認定作業に進めるか協議
- 認定（採択）は**年1回（9月頃）**、準備に約6ヶ月必要

## 3. 経営発展支援事業 → **条件付きで対象**

- 水耕栽培設備（IoTセンサー・LED照明・養液循環ポンプ）は**補助対象経費になる**
- ただし個別判断あり。見積書持参での相談推奨
- 公募時期は定めなし。市・県の予算次第
- **経営開始から2年以内**の要件あり → 早めの相談が必要
- **前提**: 認定新規就農者の認定が必要 → 兼業では取得不可のため、現状では利用不可

## 4. 廃校教室の利用 → **回答なし**

農業振興課の管轄外のため回答がなかったと思われる。教育委員会への別途問い合わせが必要。

## 影響分析

### 利用不可と判明した補助金
| 補助金 | 理由 |
|---|---|
| チャレンジ農業助成金 | 農地活用が前提。施設栽培は対象外 |
| 認定新規就農者 | 兼業不可（農業専従が条件） |
| 経営発展支援事業 | 認定新規就農者が前提 → 兼業不可で連鎖的に利用不可 |

### ポジティブな確認事項
- **水耕栽培は農業として認定される** → 他の農業系補助金・制度の根拠になる
- 経営発展支援事業の対象経費にIoT機器が含まれる → 将来SF専業化した場合に活用可能

### 残る補助金（最優先）
- **持続化補助金 第19回**（4/30締切、最大100万円）→ これが唯一の確実な選択肢
