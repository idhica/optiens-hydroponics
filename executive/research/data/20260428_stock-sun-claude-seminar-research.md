## 3. 中小企業向けAIセミナーの一般傾向

| 軸 | 相場・傾向 |
|----|-----------|
| 時間 | 60分（ランチタイム or 夕方）が主流。実演型は90分も多い |
| 価格 | リード獲得目的のウェビナーは**無料**が標準。有料公開セミナー2-5万円/人、企業内研修20-100万円/日 |
| 形式 | Zoomウェビナーが圧倒的。ライブ+後日アーカイブ配信が増加傾向 |
| 参加人数 | 50-200名規模。定員200名（クラスメソッド）が現状の標準的なZoom Webinar上限 |
| 集客チャネル | 自社LP、Peatix、connpass、SNS（X/LinkedIn）、メルマガ、SEOコラム |
| アジェンダ型 | 会社紹介(10分)+本編(40分)+QA(10分)の60分パターンが定番 |
| アフター | 限定資料DL→無料相談予約→個別商談→コンサル契約のファネル |

中小企業リスキリング助成金（経費助成率75%、賃金助成960円/時）が使えるため、有料化する場合は助成金前提設計が有利。

## 4. Optiens版セミナー設計のヒント

### 4-1. 一人法人で実施可能な構成
- **時間**: 60分（StockSun・クラスメソッドと同じ業界標準）
- **形式**: Zoomウェビナー（録画+生QA）。オペレーターなしで運用可能
- **頻度**: 月1回が現実的。同テーマを2-3パターンに分けて回し、配布資料を月替え（StockSun方式の踏襲）
- **定員**: 初期は20-30名で十分。少人数の方がQA密度が高く満足度も上がる

### 4-2. Optiensならではの差別化
1. **動くデモ連動**: `/document-editor` `/free-diagnosis` の実画面を見せる。StockSunは「事例紹介」止まり、Optiensは「今アクセスして触れる」
2. **自社運営ストーリー**: 「Claude Code × COO/CTO/CFO等の仮想チーム編成で1人法人を運営」という具体例。他社が出しにくい角度
3. **業種別カスタマイズ**: 9業種サンプル既公開を活かし「あなたの業種版を見せます」と訴求
4. **誠実トーン**: 「本質追求の鬼」型の煽り表現を避け、「リスクと限界も話す」誠実型。AI支援事業のブログで既に打ち出した路線と整合
5. **円相哲学**: 「3つの統合（提案・実装・保守）」を語る独自フレーム

### 4-3. 既存事業との連携ファネル
```
無料診断レポート(0円) → セミナー(0円) → 詳細レポート+MTG(5,000円) → 導入支援(個別見積) → 保守(月額)
```
セミナーは「無料診断」と「詳細レポート」の中間に位置づけ、無料診断ユーザー全員にメールで案内する設計が最適。

### 4-4. 商工会経由 vs オンラインのバランス
- **商工会経由（地域・対面）**: Phase A1主軸。少人数（10名程度）、初回は無料、信頼構築重視
- **オンライン（全国）**: Phase A2以降。録画資産化して配布、商工会セミナーの動画版として再利用

## 5. 推奨事項

### 5-1. MVP（最小構成）
| 要素 | 内容 |
|------|------|
| タイトル案 | 「Claude Codeで一人法人が回る｜中小企業のためのAIエージェント活用セミナー」 |
| 時間 | 60分（10+40+10） |
| 形式 | Zoomウェビナー、録画あり |
| 参加費 | 無料 |
| 定員 | 30名 |
| 配布資料 | 「業種別AIエージェント設計テンプレート9選」（既存サンプル流用） |
| 講師 | 代表1名（一人法人を「強み」として打ち出す） |
| アフター | 無料相談1枠 + 詳細レポート(5,000円)案内 |

### 5-2. 段階的拡張プラン
- **3ヶ月以内**: 月1回の無料ウェビナー定着、録画ライブラリ化
- **6ヶ月以内**: 業種特化版（士業向け／EC向け／製造業向け）の3パターン化
- **12ヶ月以内**: 有料ハンズオン版（5,000-15,000円、定員10名、helmsman型）追加。助成金活用導線を整備
- **2年目**: 自治体・商工会との共同開催（自治体冠で集客力UP）

### 5-3. 必要な準備物・ツール
- **配信**: Zoom Pro（既契約あれば流用） or Zoom Webinar（月額約20,000円）
- **LP**: Astroの既存サイトに `/seminar/` ページ追加（StockSunと同じく1セミナー1ページ）
- **申込フォーム**: Googleフォーム or 自社実装。必須6項目（StockSunと同等）
- **メール配信**: 既存の無料診断登録者リストに案内
- **資料**: スライド30-40枚（Canva or Slides）、配布PDF1本（既存9業種サンプルから抽出）
- **録画**: Zoomの自動録画+YouTube限定公開。後日「セミナー動画」としてLP掲載
