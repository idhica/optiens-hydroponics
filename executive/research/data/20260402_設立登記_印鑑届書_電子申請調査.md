# 合同会社設立登記 — 印鑑証明書・印鑑届書・会社実印の調査結果

## 1. 電子申請（マイナンバーカード電子署名）の場合、印鑑証明書は必要か？

### 結論: 代表社員個人の印鑑証明書は不要

- マイナンバーカードの電子証明書で電子署名を行うことで、市区町村長発行の**個人の印鑑証明書の添付が不要**になる
- 法務局の公式ページ（2026年2月更新）にも「電子証明書を用いて電子署名を行うことにより、一部の添付書類の提出が不要」と明記
- 電子証明書は「書面取引における印鑑証明書に代わるもの」という位置づけ

### 条件
- **完全オンライン申請**であること（書面申請との混合は不可）
- マイナンバーカード + ICカードリーダライタが必要
- 社員全員の電子証明書が必要（合同会社Optiensは代表社員1名のみなので問題なし）

## 2. 印鑑届書（会社実印の届出）について

### 2-1. 制度上の扱い（2021年2月15日改正）

- 令和3年（2021年）2月15日、商業登記規則が改正
- **オンラインで登記申請する場合に限り、印鑑届書の提出が任意**になった
- 書面（窓口・郵送）で登記申請する場合は、従来通り印鑑届書が**必須**

### 2-2. マイナポータル法人設立ワンストップサービスでの印鑑届書

- ワンストップサービスはオンライン申請なので、**印鑑届書の提出は任意**
- 印鑑届書を提出する場合でも、ワンストップサービスから代表者の印鑑証明書をオンライン送信可能
- つまり「印鑑届書を出す／出さない」を選択できる

### 2-3. 会社実印なしで法人運営は可能か？ — 実務上は困難

**印鑑届出をしない場合の最大のデメリット:**
- **法人の印鑑証明書が発行できない**（法務局に届出がないため）

**印鑑証明書が必要になる場面（実務上非常に多い）:**

| 場面 | 影響度 | 備考 |
|---|---|---|
| **法人口座開設** | 極めて高い | ほぼ全ての銀行で法人印鑑証明書を要求。GMOあおぞらネット銀行も例外ではない可能性が高い |
| **融資・借入** | 極めて高い | 金融機関は必ず印鑑証明書を要求 |
| **補助金・助成金申請** | 高い | 持続化補助金等で印鑑証明書の添付を求められることがある |
| **不動産賃貸借契約** | 高い | Phase 2の別荘3階の契約時に必要になる可能性大 |
| **許認可申請** | 高い | 行政への申請で実印押印を求められることが多い |
| **取引先との重要契約** | 中程度 | 電子契約で代替可能な場合もあるが、相手方が紙の契約書+実印を求めるケースが多い |

**司法書士の見解（複数の専門家が一致）:**
> 「制度上は任意になったが、実際には印鑑届は必須。金融機関で融資を受ける際や、行政へ許認可を申請する際などに、実印での押印を求められることがまだまだ多い」

### 2-4. 結論と推奨

**会社実印は作成し、印鑑届書は提出すべき。**

理由:
1. 法人口座開設（E-09）で確実に必要
2. 補助金申請（E-14等）で必要になる可能性が高い
3. Phase 2の賃貸借契約（E-26）で必要
4. 実印の購入コストは数千円〜1万円程度で、省略するメリットが小さい
5. 専門家も「完全オンライン申請の手間 vs 実印購入代数千円」で後者を推奨

### 3-2. 推奨サイズ・素材・価格帯

**サイズ:**
- 代表者印（実印）: **18mm**（法人実印の標準サイズ。法務局の規定: 1cm〜3cmの正方形に収まること）
- 銀行印: 16.5mm（法人口座開設用）
- 角印: 21mm（請求書・領収書用、任意）

**素材と価格帯（代表者印 単品の目安）:**

| 素材 | 価格帯 | 特徴 | 推奨度 |
|---|---|---|---|
| **薩摩本柘（つげ）** | 3,000〜5,000円 | 最安。木製で温かみがある。耐久性はやや劣る | コスト最優先なら |
| **黒水牛** | 5,000〜8,000円 | コスパ良好。耐久性と見栄えのバランスが良い | **推奨** |
| **牛角（オランダ水牛）** | 8,000〜15,000円 | 高級感あり。白〜飴色 | — |
| **チタン** | 10,000〜20,000円 | 最高耐久性。半永久的に使用可能。欠けない | 長期使用なら |

**セット購入の場合（実印+銀行印+角印）:**

| 素材 | セット価格 | 備考 |
|---|---|---|
| 柘・寸胴3本セット | 約13,000〜15,000円 | — |
| 黒水牛・天丸3本セット | 約18,000〜22,000円 | — |
| チタン・天丸3本セット | 約25,000〜35,000円 | — |

**合同会社Optiensへの推奨:**
- **最低限**: 代表者印（実印）1本のみ → **黒水牛 18mm 単品で5,000〜8,000円**
- **実用的**: 実印+銀行印の2本セット → **黒水牛で13,000〜15,000円**
- 角印は請求書・領収書に使うが、freee等のクラウドサービスで電子発行するなら不要

**刻印内容:**
- 外枠（回文）: 「合同会社Optiens」
- 内枠（中文）: 「代表社員之印」

### 3-3. 印鑑届書の提出タイミング

**設立登記と同時でなくても、後から届出可能。**

- オンライン申請の場合、設立登記時に印鑑届書を提出しなくても登記は完了する
- 印鑑届書は設立後いつでも法務局に提出可能（期限の定めなし）
- 提出方法: 管轄法務局の窓口に持参、または郵送
- 届出には代表社員個人の印鑑証明書（市区町村発行）の添付が必要
- 法務省はオンラインでの印鑑届出にも対応（商業登記電子認証ソフト使用）

## 4. 合同会社Optiensへの推奨アクション

### 即座に実行すべきこと

1. **会社実印をオンライン注文**（本日中）
   - 推奨: ハンコヤドットコムまたは印鑑本舗で黒水牛 18mm 代表者印
   - 刻印: 外枠「合同会社Optiens」/ 内枠「代表社員之印」
   - 銀行印も同時注文推奨（法人口座開設 E-09 で必要）

2. **設立登記はまずオンラインで申請**（印鑑届書なしで可）
   - マイナポータル法人設立ワンストップサービスで電子申請
   - 印鑑証明書の添付不要（マイナンバーカード電子署名で代替）

3. **実印到着後に印鑑届書を法務局に提出**（4/3〜4/4届き想定）
   - 甲府地方法務局（管轄）に窓口持参または郵送
   - 届出時に代表社員個人の印鑑証明書が必要 → 事前にコンビニ交付で取得

### タイムライン

```
4/2（木）: 会社実印をオンライン注文 + 設立登記を電子申請（印鑑届書なし）
4/3（金）: 実印到着（最短）→ 印鑑届書を法務局に提出
4/4（土）: 実印到着（遅延時）
4/6（月）: 印鑑届書を法務局に提出（実印が土曜着の場合）
4/6（月）: ★社会保険 新規適用届の期限（設立5日以内）
4/10前後: 法人口座開設申請（印鑑証明書が必要）
```
