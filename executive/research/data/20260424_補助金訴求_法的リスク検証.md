# 補助金訴求の法的リスク検証レポート

**検証日**: 2026年4月24日
**対象**: ai-consultingページ・free-diagnosis-thanksページの補助金訴求文言
**検証手法**: 悪魔の代弁者 + 前提の疑問化 + スチールマン

---

## エグゼクティブサマリー

| リスク領域 | 深刻度 | 現状の対応 | 要修正 |
|-----------|--------|-----------|--------|
| 景品表示法（有利誤認） | **中** | 条件付き表現あり | 一部修正推奨 |
| 補助金適正化法 | **低** | 問題なし | 不要 |
| 消費者契約法 | **低〜中** | 免責不足 | 注記追加推奨 |
| 不正競争防止法 | **低** | 問題なし | 不要 |
| 補助金制度変更リスク | **中** | 未対応 | 注記追加必須 |
| 委託費該当性の不確実性 | **高** | 断定的表現 | 修正必須 |

**総合判定**: 現状の文言は「違法」とまでは言えないが、**3箇所に法的リスクがあり修正を推奨**する。特に「委託費として申請可能です」の断定表現が最大のリスク。

## 1. 景品表示法（有利誤認）のリスク

### 1-1. 「実質約5万円」表現の検証

**現状の文言**:
> 採択された場合、費用の2/3が補助されます（AI活用診断15万円 → **実質約5万円**）。

**リスク評価: 中**

景品表示法第5条第2号（有利誤認表示）は「実際よりも著しく有利であると一般消費者に誤認される表示」を禁止している。

**悪魔の代弁者の論点**:
- 「実質約5万円」は太字で強調されており、閲覧者の注意を最も引く部分
- 補助金の採択率は一般に30〜50%程度であり、過半数の申請者は不採択
- 「採択された場合」という条件は付されているが、視覚的に「実質約5万円」が目立つレイアウトになっている
- 携帯電話の「実質0円」が総務省から問題視された前例がある

**スチールマン（反論の最強形）**:
- 「採択された場合」という条件を明示している
- 補助率「2/3」は制度上の事実である
- 計算結果「実質約5万円」は算術的に正確
- 消費者（事業者）は補助金制度を理解している層が多い

**判定**: 現時点で違法とは言えないが、**「実質」の表現は誤解を招くリスクがある**。総務省の携帯電話「実質0円」規制の流れを考慮すると、「自己負担額は約5万円」への変更が安全。

### 1-2. 条件付き表現の十分性

**現状の文言**:
> 小規模事業者持続化補助金の「委託費」として**申請可能です**。

**リスク評価: 高**

**悪魔の代弁者の論点**:
- 「申請可能です」は事実上の断定表現。「委託費として認められる」と読める
- **実際には、AI活用診断が委託費として認められるかは審査次第**。公募要領では「自ら実行することが困難な業務」に限定されている
- Optiensのサービスが委託費として不採択になった実績はゼロ（採択実績もゼロ）
- 顧客が「申請可能と書いてあったから申請したのに不採択だった」とクレームした場合、Optiensに説明責任が生じる

**前提の疑問化**:
- AI活用診断が「販路開拓のための委託」と審査員に認められる保証はない
- 商工会議所の担当者によって判断が分かれる可能性がある
- 第20回以降で委託費の対象範囲が変更される可能性がある

**判定**: **「申請可能です」は「申請をご検討いただけます」に変更すべき**。断定を避けることで法的リスクが大幅に低減する。

### 1-3. 補助率変更リスク

**現状の文言**: 補助率の変更可能性について言及なし。

**リスク評価: 中**

- 補助率2/3は通常枠の現行制度。今後の公募回で変更される可能性がある
- 赤字事業者は3/4だが、この点の言及もない
- 制度変更時にサイト更新が遅れると、事実と異なる表示が残る

**判定**: 「※ 補助率・対象要件は公募回により変更される場合があります」の注記を追加すべき。

## 2. 補助金関連法規のリスク

### 2-1. 補助金申請を誘導する広告の規制

**リスク評価: 低**

補助金適正化法（昭和30年法律第179号）は補助金の交付・使用に関する法律であり、**民間事業者が「この費用は補助金で使えます」と広告すること自体を規制する条文はない**。

ただし、以下の場合は問題になり得る:
- 補助金の不正受給を教唆・幇助する場合（刑法の共犯規定）
- 虚偽の内容で補助金申請を促す場合

現状の文言は制度情報の提供に留まっており、不正を促す内容ではないため問題なし。

### 2-2. 「補助金が使えます」vs「申請可能です」

**リスク評価: 中**

- 「補助金が使えます」: 採択を前提としており、不採択リスクを隠蔽。有利誤認のリスクが高い
- 「申請可能です」: 採択の不確実性は暗示しているが、申請自体の適格性を断定
- 「申請をご検討いただけます」: 最も安全。Optiensは制度情報を提供しているだけ

**業界慣行**: IT導入支援事業者（大塚商会、TIS等）のLPでは「IT導入補助金対象」「補助金を活用できます」という断定表現が一般的。ただしこれらは**IT導入支援事業者として公式に登録済み**であり、制度上の裏付けがある。

**Optiensの場合**: IT導入支援事業者に登録しておらず、持続化補助金の委託費としての採択実績もない。この状態で「申請可能」と断定するのは、登録済み事業者と同列の表現としてリスクが高い。

### 2-3. IT導入補助金への言及

**現状**: サイト上でIT導入補助金（デジタル化・AI導入補助金）には直接言及していない。

**リスク評価: 該当なし**

IT導入支援事業者でない事業者がIT導入補助金について言及すること自体に法規制はないが、「当社のサービスはIT導入補助金の対象です」と記載した場合、IT導入支援事業者としての登録がない限り虚偽表示となる。現状は言及していないため問題なし。

## 3. 不正競争防止法・消費者契約法のリスク

### 3-1. 競合との比較における優位性主張

**リスク評価: 低**

不正競争防止法第2条第1項第20号（品質等誤認惹起行為）は、商品・サービスの品質等について誤認させる表示を禁止している。

現状の文言は「補助金制度の情報提供」であり、競合との比較ではないため、不正競争防止法上の問題はない。

### 3-2. 補助金前提の価格表示

**リスク評価: 中**

消費者契約法第4条（誤認類型）は、「重要事項について事実と異なること」や「不利益事実の不告知」を規制している。

**前提の疑問化**:
- 顧客が「実質5万円」を信じてサービスを契約→補助金不採択→15万円の全額負担
- この場合、「不利益事実の不告知」に該当するか?

**判定**: 現状の文言には以下の不利益事実が不告知:
1. 補助金の採択率（全申請者の半数以上が不採択の可能性）
2. 採択されても補助金は後払い（先に15万円の全額を支払う必要がある）
3. 採択後に実績報告が必要（報告不備で補助金取消の可能性）
4. 申請から採択・入金まで数ヶ月かかる

特に**「先払い」の事実は顧客の意思決定に重大な影響を与える**ため、明記すべき。

## 4. 隠れたリスク: Optiens自身の補助金取り下げとの矛盾

**リスク評価: レピュテーション・中**

Optiens自身が持続化補助金を「報告義務が重い」「手間に見合わない」として取り下げている（2026年4月14日決定）。一方で顧客には「補助金で実質5万円」と訴求している。

**悪魔の代弁者の論点**:
- 自社が「面倒だから使わない」と判断した制度を、顧客には「使えます」と勧めている
- 顧客が補助金申請の手間（事業支援計画書取得・電子申請・実績報告）を知った場合、「そんな面倒なことは聞いていない」となるリスク
- 法的リスクではないが、信頼を損なう可能性がある

**判定**: 法的問題ではないが、**補助金申請に伴う顧客の手間・負担を明記すべき**。特に商工会議所への相談・事業計画書作成・実績報告の必要性は事前告知が望ましい。

## 5. 改善提案

### 5-1. ai-consultingページの修正案

**現状**:
```
補助金で自己負担を軽減できます

小規模事業者持続化補助金の「委託費」として申請可能です。
採択された場合、費用の2/3が補助されます（AI活用診断15万円 → 実質約5万円）。

※ 小規模事業者（従業員20名以下）が対象です。詳細はお問い合わせください。
```

**修正案**:
```
補助金の活用もご検討いただけます

AI活用診断の費用は、小規模事業者持続化補助金の「委託費」として
申請できる場合があります。
採択された場合、費用の2/3が補助され、自己負担は約5万円となります。

※ 補助金には審査があり、採択を保証するものではありません。
※ 補助金は後払いのため、サービス利用時に費用の全額をお支払いいただきます。
※ 対象は小規模事業者（商業・サービス業: 従業員5名以下、製造業等: 20名以下）です。
※ 補助率・対象要件は公募回により変更される場合があります。
※ 申請には商工会議所等での事業支援計画書の取得が必要です。詳細はお問い合わせください。
```

**変更点**:
1. 「申請可能です」→「申請できる場合があります」（断定回避）
2. 「実質約5万円」→「自己負担は約5万円」（「実質」表現の回避）
3. 採択非保証の明記
4. 後払いである事実の明記
5. 従業員基準の正確な記載（商業5名以下/製造業20名以下）
6. 制度変更可能性の注記
7. 申請手続きの負担の示唆

**現状**:
```
有償版のAI活用診断（15万円）は持続化補助金の委託費として申請可能です。
採択された場合、実質約5万円でご利用いただけます。
```

**修正案**:
```
有償版のAI活用診断（15万円）は、小規模事業者持続化補助金の委託費として
申請できる場合があります。
採択された場合、自己負担は約5万円になります（※審査があり、採択を保証するものではありません）。
```

### 5-3. 追加で検討すべき対策

| 対策 | 優先度 | 理由 |
|------|--------|------|
| 補助金情報ページの別途作成 | 低 | 詳細な制度説明・免責事項をまとめたページを用意し、LPからリンク |
| 採択実績の掲載 | 中 | 将来的に顧客の採択実績が出た場合、訴求の説得力と適法性が向上 |

## 6. 業界慣行との比較

### IT導入支援事業者の補助金訴求パターン

大塚商会・TIS・ラクス等のIT導入支援事業者は以下のパターンで訴求:

1. **「IT導入補助金対象」バッジ**: 登録済み事業者としての公式な裏付けあり
2. **具体的な補助額の明示**: 「最大450万円」等の上限額を記載
3. **免責事項の明記**: 「審査があります」「採択を保証するものではありません」
4. **申請サポートの提供**: 申請書作成支援をサービスに含める

**Optiensとの違い**: これらの事業者はIT導入支援事業者として**公式に登録済み**であり、「対象」と記載する制度上の根拠がある。Optiensは持続化補助金の委託費としての訴求であり、制度上は可能だが、**採択実績がないため断定表現は避けるべき**。

---

## 結論

現状の文言は**違法ではないが、3つの修正を強く推奨**する:

1. **「申請可能です」→「申請できる場合があります」**: 委託費該当性の不確実性を反映
2. **「実質約5万円」→「自己負担は約5万円」**: 有利誤認リスクの低減
3. **免責注記の追加**: 採択非保証・後払い・制度変更可能性の明記

これらの修正により、景品表示法・消費者契約法上のリスクを実質的に解消できる。修正は軽微であり、訴求力の低下も最小限に抑えられる。
