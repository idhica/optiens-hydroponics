# 水耕栽培ハーブ ユニットエコノミクス算出方法

**調査日**: 2026年3月25日

---

## エグゼクティブサマリー

1. **バジル1株あたりの生産コストはラック1段（20株）で約70-80円、4段フル稼働（80株）で約20-30円（Phase 1試算値）**。最大コスト項目はLED電気代（40-50%）。栽培株数を増やすことで固定費（電気代・減価償却）の按分が進み、大幅にコスト低減可能
2. **損益分岐点は「1株あたり生産コスト < 販売単価の40%」**が業界の健全ライン。バジル1株50g収穫×レストラン直販4,000円/kg=200円/株の場合、生産コスト80円以下が目標
3. **Phase 1で計測すべき最優先KPIは「1株あたり生産コスト」「kWh/kg（電力原単位）」「収穫サイクル日数」の3つ**。これらが確立できなければPhase 2に進むべきではない

## 1. バジル1株あたりコスト構造

### 1-1. コスト項目の分解

| コスト項目 | 内容 | 1段20株の場合 | 4段80株の場合 | 出典 |
|---|---|---|---|---|
| **電気代（LED照明）** | フルスペクトルLED、16h/日点灯 | 34円 | 8円 | 試算値（後述の計算式参照） |
| **電気代（ポンプ・RPi等）** | ポンプ5W+RPi×2=15W、24h稼働 | 15円 | 4円 | 試算値 |
| **養液代** | ハイポニカ液肥A液+B液 | 6円 | 3円 | 試算値（後述の計算式参照） |
| **種子代** | スイートバジル種 | 1.5円 | 1.5円 | バジル種子300円/袋（約200粒）=1.5円/粒 |
| **水代** | 水道水 | 1円 | 1円 | 水道料金約0.2円/L |
| **人件費（時間換算）** | 播種・移植・収穫・パッキング | 5円 | 5円 | 時給1,000円、1株0.3分（試算値） |
| **減価償却費** | LED・ラック・センサー・ポンプ | 26円 | 7円 | 試算値（後述の計算式参照） |
| **その他（消耗品）** | スポンジ培地・包装資材 | 2円 | 2円 | スポンジ2円/個 |
| **合計** | | **約91円** | **約31円** | 試算値 |

> **注意**: 1段20株は初期テスト時の値。4段80株はPhase 1後半（ラック全段稼働時）の値。LED・ポンプ・RPiの電力は固定費のため、栽培株数が増えるほど1株あたりコストが大幅に低下する。Phase 2移行判断は4段フル稼働時の数値で行う。

### 1-2. 電気代の詳細計算式

```
LED電気代/株 = (消費電力W × 点灯時間h × 栽培日数 × 電気単価円/kWh) / 同時栽培株数

【Phase 1の具体例】
- LEDパネル: 50W（保有品）
- 点灯時間: 16時間/日
- 栽培日数: 28日（播種→収穫）
- 電気単価: 30円/kWh（山梨県平均、出典: 東京電力EP従量電灯B第3段階）
- 同時栽培株数: 20株（育苗トレイ×2、ラック1段）

計算: (0.05kW × 16h × 28日 × 30円) / 20株 = 33.6円/株

※ポンプ・センサー等の電力は別途加算（約2-5円/株）
```

**重要**: LED照明の効率（umol/J）が電気代を決定的に左右する。

### 1-3. 養液代の詳細計算式

```
養液代/株 = (液肥価格 / 希釈倍率 × 1株あたり養液消費量L) × 栽培日数における養液交換回数

【Phase 1の具体例】
- ハイポニカ液肥: A液500mL + B液500mL = 約1,500円/セット
- 希釈倍率: 500倍（A液1mL + B液1mL → 1L養液）
- 養液コスト: 約3円/L
- 20Lタンク1回分: 約60円
- 養液交換: 2週間に1回（28日栽培で2回）
- 20株で共有: (60円 × 2回) / 20株 = 6円/株
```

### 1-4. 減価償却費の計算式

```
減価償却費/株 = 設備総額 / (耐用年数 × 年間栽培株数)

【Phase 1の具体例】
- 設備総額（保有品の再調達価格で試算）:
  - LEDパネル: 5,000円
  - メタルラック: 4,000円
  - RPi 4B ×2: 16,000円
  - センサー類一式: 8,000円
  - ポンプ・配管: 7,575円
  - 合計: 約40,575円
- 耐用年数: 加重平均6年
- 年間栽培株数: 20株 × 13サイクル/年 = 260株/年

計算: 40,575円 / (6年 × 260株) = 約26円/株

※Phase 1は設備コストの回収を目的としない（テスト期間）
```

## 2. コスト項目の計測方法・計測頻度

| コスト項目 | 計測方法 | 計測頻度 | 必要ツール |
|---|---|---|---|
| **電気代** | スマートプラグの消費電力ログ（Zigbeeスイッチ経由でMQTT記録） | **リアルタイム**（日次集計） | Zigbeeスマートスイッチ（電力計測機能付き） |
| **養液代** | タンク補充量の記録（液肥使用量mL × 単価） | **養液交換時**（2週間に1回） | 計量カップ、記録シート |
| **種子代** | 播種数の記録（発芽率も記録） | **播種時**（サイクルごと） | 記録シート |
| **水代** | 水道メーター読み取り or タンク補充量 | **月次** | 水道メーター |
| **人件費** | 作業時間のストップウォッチ計測（作業種別ごと） | **作業ごと** | スマホタイマー、記録シート |
| **減価償却費** | 購入価格 / 耐用年数で月次按分 | **月次**（固定値） | 購入記録 |
| **収穫量** | デジタルスケールで計量（株数も記録） | **収穫時** | デジタルスケール（0.1g精度） |

### 計測の優先順位（Phase 1）

1. **電気代**: 最大コスト項目。Zigbeeスイッチで自動ログ化が最優先
2. **収穫量**: 1株あたりグラム数が不明では全ての計算が成立しない
3. **作業時間**: Phase 2の人件費見積もりに直結
4. **養液消費量**: 液肥コストとEC/pH管理の両方に影響

## 5. 損益分岐点の計算式

### 5-1. 基本公式

```
損益分岐点（株数/月） = 月間固定費 / (1株あたり売上 - 1株あたり変動費)
```

### 5-2. Phase 1の損益分岐点試算

```
【固定費（月額）】
- LED電気代: 50W × 16h × 30日 × 30円/kWh = 720円
- ポンプ電力: 5W × 24h × 30日 × 30円/kWh = 108円
- RPi電力: 5W × 2台 × 24h × 30日 × 30円/kWh = 216円
- 減価償却: 40,575円 / 72ヶ月 = 564円
- 合計: 約1,608円 → 余裕を見て約2,000円

【変動費（1株あたり）】
- 種子: 1.5円、養液: 3円、水: 1円、培地: 2円、包装: 1円、人件費: 5円
- 合計: 約13.5円

【1株あたり売上（レストラン直販想定）】
- 収穫量50g × 4円/g（4,000円/kg） = 200円

損益分岐点 = 2,000円 / (200円 - 13.5円) = 約11株/月
→ 80株/サイクル（4段フル稼働）なら月2サイクル以上で十分に上回る
```

### 5-3. Phase 2の損益分岐点試算

```
- 月間固定費: 約80,000円（家賃+パート人件費+電気代+その他）
- 1株あたり変動費: 約10円
- 1株あたり売上: 200円

損益分岐点 = 80,000円 / (200円 - 10円) = 約422株/月（≒21.1kg）
→ 20m²で月間25-40kg生産可能のため達成可能だが余裕は少ない
```

### 5-4. 業界の健全性指標

| 指標 | 健全ライン | 危険ライン | 出典 |
|---|---|---|---|
| 生産コスト/販売単価 | 40%以下 | 60%超 | 既存調査 |
| エネルギーコスト/総収益 | 20%未満 | 30%超 | 既存調査 |
| 貢献利益率 | 80%以上 | 60%未満 | 既存調査 |
| ASP/CPU比率 | 2.5倍以上 | 1.5倍未満 | 既存調査 |

### 6-2. サイクル記録シート（播種〜収穫で1レコード）

| 記録項目 | 単位 | 備考 |
|---|---|---|
| サイクルID | - | 例: C001 |
| 品種 | - | スイートバジル / マイクログリーン等 |
| 播種日 | YYYY-MM-DD | |
| 播種数 | 株 | |
| 発芽数 | 株 | 発芽率 = 発芽数/播種数 |
| 収穫日 | YYYY-MM-DD | |
| 収穫量 | g | デジタルスケールで計量 |
| 栽培日数 | 日 | 播種→収穫 |
| 総電力消費量 | kWh | LED+ポンプ合計 |
| 養液使用量 | mL（原液） | A液+B液の合計 |
| **1株あたり総コスト** | 円 | 自動計算 |
| **1gあたりコスト** | 円 | 自動計算 |
| **kWh/kg** | - | 電力原単位。自動計算 |

### 6-3. 月次サマリーシート

| 記録項目 | 計算方法 |
|---|---|
| 月間総収穫量(g) | サイクルシートから集計 |
| 月間総コスト(円) | 全コスト項目の合計 |
| 月間平均1株コスト(円) | 総コスト / 収穫株数 |
| kWh/kg | 月間電力 / 月間収穫量(kg) |
| エネルギーコスト比率(%) | 電気代 / 仮想売上 × 100 |

### 6-4. 実装方法

- **データ格納先**: Supabase（IoTセンサーデータと同じDB）
- **自動入力**: 電力・環境データはMQTT→Supabaseで自動記録
- **手動入力**: 播種数・収穫量・作業時間はスマホからフォーム入力
- **Phase 1初期**: Googleスプレッドシートで手動管理も可（IoTシステム安定化前の暫定措置）

## 7. 成功企業が追跡しているKPI一覧

### 7-1. 財務KPI

| KPI | 定義 | ベンチマーク | 計測頻度 |
|---|---|---|---|
| **粗利益率** | (売上 - 変動費) / 売上 × 100 | 60%以上（ハーブ） | 月次 |
| **1株あたり生産コスト（CPU）** | 全コスト / 株数 | ASPの40%以下 | サイクルごと |
| **ASP/CPU比率** | 平均販売単価 / 1株コスト | 2.5倍以上 | 月次 |
| **エネルギーコスト比率** | 電気代 / 売上 × 100 | 20%未満 | 月次 |

### 7-2. 生産KPI

| KPI | 定義 | ベンチマーク | 計測頻度 |
|---|---|---|---|
| **m²あたり年間収量** | 年間収穫量(kg) / 栽培面積(m²) | 葉物: 20-30kg/m²/年 | 月次 |
| **収穫サイクル日数** | 播種→収穫の日数 | バジル: 28日以内 | サイクルごと |
| **発芽率** | 発芽数 / 播種数 × 100 | 85%以上 | サイクルごと |
| **歩留まり率** | 出荷可能株数 / 播種数 × 100 | 75%以上 | サイクルごと |
| **収穫量変動係数（CV）** | 標準偏差 / 平均 × 100 | 20%以内 | 月次 |

### 7-3. 運営効率KPI

| KPI | 定義 | ベンチマーク | 計測頻度 |
|---|---|---|---|
| **kWh/kg（電力原単位）** | 消費電力(kWh) / 収穫量(kg) | 低いほど良い | サイクルごと |
| **作業効率** | 収穫量(g) / 作業時間(h) | 高いほど良い | 月次 |
| **システム稼働率** | 稼働時間 / 計画稼働時間 × 100 | 95%以上 | 月次 |

### 7-4. Phase 1で追跡すべきKPI（優先順位）

| 優先度 | KPI | 理由 |
|---|---|---|
| **1** | 1株あたり生産コスト（CPU） | 全経営判断の基盤 |
| **2** | kWh/kg（電力原単位） | 最大コスト項目の効率指標 |
| **3** | 収穫サイクル日数 | キャッシュフロー速度の決定要因 |
| **4** | 1株あたり収穫量(g) | 販売単価計算の基礎 |
| **5** | 発芽率・歩留まり率 | 計画精度に直結 |

## 8. 要確認事項

1. **保有Zigbeeスマートスイッチの電力計測機能の有無** — 電力ログが取れない場合、ワットチェッカー（約1,500円）の追加購入を検討
2. **デジタルスケールの保有状況** — 0.1g精度のキッチンスケールが必要
3. **北杜市の電気料金の正確な単価** — 東京電力EP第3段階（約30円/kWh）で試算
4. **ハイポニカ液肥の正確な希釈倍率と価格** — 保有品のロットで変動の可能性
