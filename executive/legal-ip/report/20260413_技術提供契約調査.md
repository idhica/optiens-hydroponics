## 1. 契約類型の整理

### 1.1 合同会社Optiensが必要とする契約体系

Optiensの技術提供事業（Phase 5〜）では、以下の5種類の契約が必要となる。

| # | 契約名 | 法的性質 | 締結先 | 概要 |
|---|---|---|---|---|
| 1 | 技術コンサルティング契約 | 準委任（民法656条） | 導入検討先 | 設計相談・現地調査・導入計画策定 |
| 2 | システム導入支援契約 | 請負（民法632条） | 導入決定先 | ハードウェア納品・設置・初期設定・稼働確認 |
| 3 | 保守サポート契約 | 準委任（民法656条） | 稼働中の顧客 | 月額制の遠隔監視・障害対応・栽培アドバイス |
| 4 | 営業パートナー契約 | 代理店契約（商法27条等） | 福祉コンサルタント等 | 紹介・取次に対するコミッション支払い |
| 5 | 秘密保持契約（NDA） | 契約自由の原則 | 全取引先 | 商談開始前に締結 |

### 1.2 準委任 vs 請負の使い分け

| 比較項目 | 準委任 | 請負 |
|---|---|---|
| 目的 | 業務遂行そのもの（プロセス） | 仕事の完成（成果物） |
| 成果物の要否 | 不要（報告書等は任意） | 必要（納品・検収） |
| 瑕疵担保 | なし（善管注意義務） | あり（契約不適合責任） |
| 報酬発生条件 | 業務遂行により発生 | 完成・引渡しにより発生 |
| 中途解約 | 委任者はいつでも可（損害賠償義務あり） | 注文者はいつでも可（損害賠償義務あり） |
| Optiensでの適用 | コンサルティング、保守サポート | システム導入（納品・検収あり） |

**判断基準**: 「システムが動く状態で引き渡す」ことを約束する導入支援は請負、「助言・監視・対応を行う」コンサルティングと保守は準委任が適切。

## 2. 各契約に含めるべき重要条項

### 2.1 全契約共通条項

| 条項 | 内容 | 備考 |
|---|---|---|
| 秘密保持 | 契約期間中および終了後3年間 | NDAと整合させる |
| 個人情報保護 | 個人情報保護法に基づく取り扱い | 栽培データに個人情報が含まれる場合 |
| 反社会的勢力排除 | 暴排条項（表明・保証・解除権） | 東京都暴排条例等に準拠 |
| 権利義務の譲渡禁止 | 書面による事前同意なく譲渡不可 | |
| 不可抗力 | 天災・感染症・法令変更等による免責 | |
| 管轄裁判所 | 甲府地方裁判所（Optiens本店所在地） | 山梨県北杜市の管轄 |
| 協議解決 | 紛争は誠実に協議、解決しない場合は裁判 | |
| 準拠法 | 日本法 | |

### 2.2 知的財産権の帰属（最重要）

Optiensのビジネスモデル上、以下の区分が不可欠。

| 対象 | 帰属先 | 理由 |
|---|---|---|
| システム設計図・回路図 | Optiens | コア技術。複数顧客に展開するため |
| 制御ソフトウェア・ファームウェア | Optiens | ライセンス提供形態 |
| 栽培アルゴリズム・AIモデル | Optiens | 事業の根幹。ノウハウの蓄積 |
| 栽培データ（環境データ・収穫データ） | 顧客 | 顧客の事業活動から生じるデータ |
| 匿名化・集約済みデータ | Optiens | アルゴリズム改善に必要（顧客同意前提） |
| 顧客固有のカスタマイズ部分 | 共有（要協議） | 顧客の要望による改変 |

**重要**: 契約書で「Optiensが従前から保有するIPは全てOptiens帰属」と明記し、導入時に生じた改良もOptiens帰属とする条項を入れる。顧客にはシステムの「利用権」を許諾する形式が望ましい。

### 2.3 免責事項

以下の事象による損害についてOptiensは責任を負わない旨を明記する。

- 天災・地震・水害・落雷
- 停電・電力障害（顧客施設の電力供給に起因するもの）
- 顧客による不適切な操作・改造・未承認の設定変更
- 第三者製品（種子・肥料・水質等）に起因する栽培不良
- 通信障害（インターネット回線・クラウドサービスの障害）
- 栽培結果の保証（収穫量・品質の保証は行わない）

**注意**: 「収穫量を保証しない」ことは特に重要。水耕栽培は多くの外部要因に影響されるため、結果保証は法的リスクが高い。

### 2.4 契約不適合責任（旧瑕疵担保責任）

| 契約類型 | 責任範囲 | 期間 | 対応 |
|---|---|---|---|
| システム導入支援（請負） | 納品物の仕様不適合 | 検収完了後6ヶ月 | 修補または代替品提供 |
| 保守サポート（準委任） | 善管注意義務違反のみ | 契約期間中 | 損害賠償（上限あり） |
| コンサルティング（準委任） | 善管注意義務違反のみ | 契約期間中 | 損害賠償（上限あり） |

**損害賠償の上限**: 当該契約の対価の総額（または直近12ヶ月分の報酬額）を上限とする。間接損害・逸失利益は賠償対象外とする。

### 2.5 中途解約・違約金

| 契約類型 | 解約条件 | 違約金 |
|---|---|---|
| コンサルティング | 1ヶ月前書面通知 | なし（未払い報酬の精算のみ） |
| 導入支援 | 着手後の解約は既発生費用を負担 | 進捗に応じた精算（材料費+工賃実費） |
| 保守サポート | 3ヶ月前書面通知 | 最低契約期間（12ヶ月）内の解約は残月分の50% |
| 営業パートナー | 3ヶ月前書面通知 | なし（既発生コミッションは支払い） |

### 2.6 再委託・第三者開示

- 再委託: Optiensの書面による事前承認を要する（設置工事の一部等）
- 再委託先の行為はOptiensの責任とする（下請法の観点からも重要）
- 第三者への技術情報開示は厳禁（NDAの範囲）

## 3. 金額設定の調査と提案

### 3.1 業界相場の調査結果

#### ITコンサルティング費用の相場

| 形態 | 相場 | 出典 |
|---|---|---|
| 時間単価 | 1.5万〜5万円/時間 | 中小企業向けITコンサル |
| 月額顧問 | 3万〜30万円/月 | 月数回の訪問・助言 |
| スポット（現状分析） | 30万〜100万円 | 現地調査+報告書 |
| パッケージ導入支援 | 150万〜500万円 | 設計〜稼働まで一括 |

#### システム保守費用の相場

| 形態 | 相場 | 備考 |
|---|---|---|
| 年間保守料 | 開発費の10〜20% | 業界標準 |
| 月額保守 | 3万〜20万円 | 規模・対応範囲による |
| 24時間対応追加 | +月額5万〜15万円 | IoTシステムでは推奨 |

#### 代理店コミッション率の相場

| 形態 | 相場 | 備考 |
|---|---|---|
| 紹介型（取次） | 10〜20% | 紹介のみで成約後は介在しない |
| 代理店型（販売） | 20〜40% | 営業・契約まで代行 |
| 継続報酬（月額サービス） | 5〜15% | ストック収入の分配 |

### 3.2 Optiensの価格設定（提案）

現行の事業計画（導入費150万円/拠点、月額サポート4万円/拠点）を基に、以下の価格体系を提案する。

#### 技術コンサルティング

| メニュー | 価格（税別） | 内容 |
|---|---|---|
| 初回相談 | 無料（1時間） | 事業概要ヒアリング・概算提示 |
| 現地調査+導入計画書 | 15万円 | 施設調査・環境測定・設計提案書作成 |
| 月額顧問（Phase 5以降） | 5万円/月 | 月2回のオンライン相談・レポート |

**注**: 導入に至った場合、コンサルティング費用は導入費から控除する運用が顧客獲得に有効。

#### フルパッケージ導入支援

| 項目 | 価格（税別） | 内容 |
|---|---|---|
| 基本パッケージ | 150万円 | 設計・機器調達・設置・初期設定・稼働確認・操作研修 |
| 含まれる機器 | (内訳) | ラック・LED・ポンプ・センサー・制御ユニット・ソフトウェア |
| オプション: 追加ラック | 15万円/台 | 2台目以降 |
| オプション: カメラ監視 | 5万円 | IPカメラ+遠隔閲覧設定 |

**内訳の目安**（原価率管理用、契約書には記載しない）:
- ハードウェア原価: 40〜50万円（粗利率60〜67%）
- 設計・設置工賃: 30万円相当
- ソフトウェアライセンス: 含む（月額サポートに継続利用権を紐付け）
- 利益: 70〜80万円

#### 月額保守サポート

| プラン | 月額（税別） | 内容 |
|---|---|---|
| スタンダード | 4万円 | 遠隔監視・メール/チャットサポート（営業時間内）・月次レポート・ソフトウェアアップデート |
| プレミアム | 7万円 | スタンダード+電話サポート+四半期訪問+栽培最適化提案 |

**最低契約期間**: 12ヶ月（初年度）。2年目以降は自動更新（3ヶ月前通知で解約可）。

#### 営業パートナーコミッション

| 対象 | コミッション率 | 備考 |
|---|---|---|
| 導入費（150万円） | 20%（30万円） | 成約時一括払い |
| 月額サポート | 10%（4,000円/月） | 契約継続中は毎月支払い |

現行事業計画と整合。業界相場（紹介型10〜20%）の上限に設定しており、パートナーのインセンティブとしては十分。ただし、パートナーが営業活動（提案書作成・デモ同行等）まで行う場合は、導入費25〜30%への引き上げも検討に値する。

### 3.3 価格設定の妥当性評価

| 指標 | Optiens | 業界相場 | 評価 |
|---|---|---|---|
| 導入費 | 150万円 | 150〜500万円 | 相場下限。就労支援事業所の予算規模に適合 |
| 月額保守 | 4万円 | 3〜20万円 | 相場内。開発費比で年間32%相当（やや高め） |
| コミッション（導入費） | 20% | 10〜40% | 紹介型としては妥当 |
| コミッション（月額） | 10% | 5〜15% | 妥当 |

**結論**: 現行の価格設定は業界相場内であり、就労支援事業所・自治体の予算規模を考慮すると適切。月額保守4万円は開発費比率では高めだが、IoTシステムの遠隔監視・栽培アドバイスを含む付加価値を考えれば正当化可能。

## 4. 就労支援事業所特有の注意点

### 4.1 A型・B型事業所の法的位置づけ

| 項目 | A型（就労継続支援A型） | B型（就労継続支援B型） |
|---|---|---|
| 根拠法 | 障害者総合支援法 | 障害者総合支援法 |
| 利用者との関係 | 雇用契約あり（最低賃金適用） | 雇用契約なし（工賃） |
| 事業収入 | 自立支援給付+事業収入 | 自立支援給付+事業収入 |
| 契約主体 | 事業所（社会福祉法人・NPO・株式会社等） | 同左 |
| 意思決定 | 理事会・管理者の承認 | 同左 |

### 4.2 契約上の注意点

1. **契約主体の確認**: 法人格の確認が必要。社会福祉法人・NPO法人・株式会社など運営母体によって意思決定プロセスが異なる。理事会承認が必要な場合、契約締結までに時間を要する。

2. **利用者（障害者）の安全配慮**:
   - 栽培作業の安全性に関する説明義務
   - 操作マニュアルの分かりやすさ（ユニバーサルデザイン）
   - 緊急時の対応手順の整備
   - 契約書に「利用者の安全に関するOptiens側の責任範囲」を明記

3. **工賃・賃金への影響の説明**:
   - B型事業所は「工賃向上」が行政評価の重要指標
   - システム導入による工賃向上の見込みを導入提案書に含めることが有効
   - ただし、工賃向上の「保証」は行わない（免責条項に含める）

4. **行政への届出**:
   - 事業内容の変更（水耕栽培の追加）に伴い、指定権者（都道府県・政令市）への届出が必要な場合がある
   - Optiensとしては契約書に「必要な届出は顧客の責任で行う」旨を明記

5. **会計処理の特殊性**:
   - 就労支援事業所は「就労支援事業会計」と「法人本部会計」が分かれている
   - 導入費・月額費用がどの会計区分に計上されるかは顧客側の判断だが、請求書の宛名・品目を適切に記載する必要がある

### 4.3 補助金活用時の契約条件

就労支援事業所が補助金（IT導入補助金・自治体独自の補助金等）を活用してOptiens製品を導入する場合の注意点。

1. **補助金の対象範囲の確認**:
   - ハードウェア・ソフトウェア・導入支援費のどこまでが対象か
   - 月額サポート費用が対象になるか（通常は初年度のみ、または対象外）
   - 見積書・請求書の区分を補助金申請に合わせて作成する必要がある

2. **契約書への記載事項**:
   - 「本契約の代金の一部または全部に補助金が充当される場合、甲（顧客）は補助金交付決定前に本契約を締結しないものとする」
   - 補助金不交付の場合の契約解除条項（白紙解除 or 自費での続行を選択）
   - 補助金の処分制限期間中（通常5年）の機器処分制限への言及

3. **事業化状況報告への協力**:
   - 補助金によっては導入後3〜5年の事業化状況報告が義務
   - Optiensとして報告に必要なデータ提供に協力する旨を契約書に含める（ただし追加費用なし）

4. **価格の適正性**:
   - 補助金審査では見積もりの妥当性が審査される
   - 相見積もり対応が必要な場合がある
   - 定価表（カタログ価格）の整備が望ましい

### 4.4 福祉サービス事業者特有の契約慣行

- **長期的な関係性を重視**: 短期的な売り切りではなく、継続的なサポート体制が求められる
- **理事会・行政への説明責任**: 導入効果の可視化（数値レポート）が重要
- **予算サイクル**: 4月始まりの年度予算。大型投資は前年度の予算編成期（10〜12月）に提案する必要がある
- **入札・プロポーザル**: 自治体が運営する事業所や、自治体の補助を受ける場合、入札・プロポーザル方式が求められることがある

## 5. 知財ポートフォリオ戦略との連携

### 5.1 契約書と知財保護の関係

技術提供契約は知財保護の「第一防衛線」として機能する。特許出願前の段階では、契約上の秘密保持義務と知財帰属条項がOptiens技術を守る唯一の手段である。

| 保護手段 | 対象 | 契約書での対応 |
|---|---|---|
| 特許 | 制御アルゴリズム・システム構成 | 出願前は秘密保持、出願後は特許番号を契約書に記載 |
| 営業秘密 | 栽培ノウハウ・パラメータ設定 | NDA+利用範囲の制限 |
| 著作権 | ソフトウェア・マニュアル | 帰属をOptiensに明記 |
| 商標 | Optiens・製品名 | 使用許諾の範囲を明記 |

### 5.2 推奨アクション

1. **Phase 4（パイロット導入前）までに**:
   - NDAテンプレートの弁護士レビュー完了
   - 主要3契約（コンサル・導入・保守）の弁護士レビュー完了
   - 営業パートナー契約の弁護士レビュー完了

2. **Phase 5（本格展開）までに**:
   - 定価表・カタログの整備
   - 補助金対応の見積書フォーマット整備
   - 電子契約サービスの導入検討（クラウドサイン等）
