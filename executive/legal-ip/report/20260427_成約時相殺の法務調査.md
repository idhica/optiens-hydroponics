# 成約時相殺（¥30,000有償要件定義 → 本契約初期費用充当）法務スクリーニング

作成日: 2026-04-27 / 担当: CLO代行（アンドレイ・ジューコフ長期休暇中のため一次調査のみ）
注意: 本書は**正式な法律相談ではなく初期スクリーニング**です。最終確定前に弁護士確認を推奨します。

---

## サマリ（結論先出し）
1. 「成約時に初期費用へ充当」スキームは**B2B取引であれば原則有効**。法的には民法上の「相殺」ではなく「**充当（前払金処理）**」として整理するのが安全。
2. 個人事業主・一人法人を顧客とする場合、**フリーランス新法（2024年11月施行）の取引条件書面明示義務**に注意。書面・電磁的記録での事前明示が必須。
3. 不成約時の¥30,000は「役務提供完了済の対価」として**返金不要を明記**することでトラブル回避可。
4. 消費税は要件定義の役務提供完了時点で課税。充当しても**減額ではなく前受金処理**が基本。
5. 推奨: 「**有償要件定義契約書**」を独立させ、第X条で「本契約締結時は本サービス報酬の一部に充当」と明記。サイレント運用は資産化できないので卒業。

## 1. 法務観点

### 1-1. 「成約時相殺」の法的有効性
- 契約自由の原則（民法521条）により、B2B間で**当事者合意があれば有効**。
- ただし厳密には民法505条の「相殺」（双方の対立債権を消す）ではなく、**前払金の本契約代金への充当**という構成が実態に即する。契約書では「相殺」という単語より「**充当する**」「**初期費用に振り替える**」が誤解を招きにくい。
- 顧客が消費者（個人）の場合は**消費者契約法10条（不当条項）**に該当しないか要注意だが、Optiensの想定顧客（法人・個人事業主）であれば原則対象外。

### 1-2. 個人事業主・小規模法人を顧客とする場合の留意点
- **フリーランス新法（特定受託事業者に係る取引の適正化等に関する法律）**: 2024年11月施行。Optiens（特定業務委託事業者）が個人事業主等に業務委託する側になる場面では適用されるが、**Optiensが受託する側**であれば直接の遵守義務は限定的。
- ただし顧客（一人法人・個人事業主）に対しても**取引条件の書面明示**（業務内容・報酬額・支払期日・充当条件）はベストプラクティス。メール・PDF送付で足りる。
- 下請法は資本金1,000万円超の発注者が対象なのでOptiensは現状対象外。

### 1-3. 不成約時の¥30,000の取り扱い
- 要件定義は**準委任契約**（民法656条）として整理するのが業界慣行。役務提供完了時点で報酬請求権が発生。
- 不成約時の返金不要を担保するには、契約書に以下を明記:
  - 「本要件定義業務は準委任契約とし、業務完了をもって報酬請求権が発生する」
  - 「本契約に至らなかった場合でも、既に支払われた要件定義費用は返金しない」
- 弁護士業界の実例では「**いかなる理由があっても返還を求めることが出来ず、返還を要しない**」型の条項が一般的（出典: 西野法律事務所、ひばり法律事務所）。

### 1-4. 「相殺」と「充当」の違い
| 概念 | 意味 | 推奨表現 |
|---|---|---|
| 相殺 | 双方が対立債権を持つ場合に対当額で消滅させる | ❌ 本件には不適 |
| 充当 | 既払金を後の債務の支払いに振り替える | ✅ 「初期費用の一部に充当する」 |
| 値引き | 本契約代金そのものを減額する | △ 消費税処理が複雑化 |

### 1-5. 消費税の扱い
- 要件定義の役務提供完了時点で課税売上が立つ（10%）。
- 本契約締結時に「充当」する場合、**前受金から本契約代金への振替**として処理すれば、要件定義分は二重課税にならない。
- 「値引き」と表現すると本契約代金そのものを減額する解釈になり、要件定義分の課税済売上の処理と齟齬が生じる可能性。**「充当」表現を強く推奨**。
- 詳細仕訳は税理士確認推奨（要件定義: 売上計上→本契約締結時: 前受金振替）。

## 2. 契約慣行・記載例

### 2-1. 業界事例
- 弁護士業界の「**着手金 → 成功報酬充当**」が最も近い類型。「着手金は返還しないが、成功報酬から控除する」運用が標準。
- ITコンサル業界では「**PoC費用の本契約充当**」「**要件定義フェーズの本開発充当**」が散見される（出典: ＠IT IT訴訟解説、咲くやこの花法律事務所）。
- 類型整理:
  - **事前金型（ノンリファンダブル）**: 返金なし・充当なし。最もシンプル。
  - **成功報酬充当型**: 不成約時は返金なし、成約時のみ本契約代金へ充当。← **Optiens想定**
  - **完全返金型**: 不成約時に返金。リスク高くOptiensには非推奨。

### 2-2. 推奨条項例（要件定義契約書）
```
第X条（要件定義費用の充当）
1. 甲（顧客）は乙（Optiens）に対し、要件定義業務の対価として金30,000円
   （消費税別）を契約締結時に支払う。
2. 本要件定義業務終了後、甲乙間で別途AI導入支援業務に関する本契約が
   締結された場合、前項の費用は本契約の初期費用の一部に充当する。
3. 前項の本契約が締結されなかった場合、第1項の費用は返金しない。
   本要件定義業務は準委任契約として、業務完了時に報酬請求権が確定する。
```

## 3. 推奨アクション

1. **契約書の二段構成化**:
   - 「**要件定義業務委託契約書**」（短文・準委任・¥30,000）
   - 「**AI導入支援業務委託契約書**」（本契約・成果物・初期費用＋月額）
   - 前者の充当条項で両者を接続。
2. **顧客への説明文（提案メール例）**:
   > 月2件以上のお問い合わせをいただいているため、無料ヒアリングの代わりに**有償要件定義（¥30,000・税別）**として実施しております。ご導入が決定した場合、本契約の初期費用に**全額充当**いたしますので、実質的なご負担は発生しません。不成約の場合は要件定義レポートをお手元に残せる形でご納品いたします。
3. **サイレント運用の卒業**: 「値引きしたかのように見せる」運用は会計・税務・将来の標準化のいずれにとっても不利。明示条項化し**会社資産（テンプレート）として蓄積**する。
4. **要弁護士確認事項**:
   - 充当時の消費税処理の最終整理（税理士同席推奨）
   - サイト掲載時の特商法表記（B2B限定であることの明記）
   - 不成約時の成果物（要件定義レポート）の著作権帰属
