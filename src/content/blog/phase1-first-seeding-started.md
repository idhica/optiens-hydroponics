## Phase 1、始動

2026年4月1日の会社設立から約2週間。本日2026年4月13日、合同会社Optiensの**Phase 1：テスト栽培**が正式に始動しました。

午前中にメタルラックへのシステム一式の設置を完了し、午後には初回播種（バジル・マイクログリーン）を実施。センサーから最初のデータがSupabaseに記録された瞬間、「農業×テクノロジーの実証実験」がついに動き始めました。

このブログでは今後、週次・月次の栽培レポートを公開していきます。データを隠さず公開することで、同じ挑戦をしている方々の参考になれば幸いです。

## 設置したシステム構成

今回稼働を開始したのは、4段メタルラック1台を核とした**循環式NFT（薄膜水耕）システム**です。

| 要素 | 詳細 |
|---|---|
| 栽培ラック | 4段メタルラック（幅90cm×奥行45cm×高さ180cm） |
| 照明 | フルスペクトルLED植物育成ライト |
| 養液循環 | DC12Vポンプ + 3段ドリップフィルター + 20L養液タンク |
| センサー | EC/pH/水温/気温湿度/CO2/照度（6種類） |
| 制御基盤 | Raspberry Pi 4B ×2台、Zigbeeスマートスイッチ |
| データ基盤 | MQTT（Mosquitto）→ Supabase |
| AI連携 | Claude Code × MCP（iPhoneからの音声操作対応） |

IoT制御により、LEDの点灯・消灯（16時間ON/8時間OFF）とポンプの間欠運転（15分ON/45分OFF）は完全自動化しています。手動作業が必要なのは、播種・収穫・養液補充の3工程のみです。

設置費用の詳細は今後の記事で公開予定ですが、センサー類・制御系を含めたシステム一式で**初期投資15万円以内**に収めることを目標にしています。

## 初回播種品目：バジルとマイクログリーン

Phase 1では2品目に絞ってテスト栽培を行います。品目選定の基準は「高単価」「短サイクル」「業務用需要の安定性」の3点です。

### バジル（スイートバジル）

- **播種数**: 48穴ウレタン培地 × 1トレイ（48株）
- **発芽予測**: 播種から5〜7日後
- **初回収穫予測**: 播種から28〜35日後
- **目標販売価格**: 50g/パック 300〜400円（業務用EC・地元レストラン）

水耕栽培バジルは、土耕と比べて**成長速度が約1.5〜2倍**になるとされています（養液から直接栄養を吸収できるため）。この数値をOptiensの環境で実際に検証します。

また、バジルは1株から複数回の収穫（摘み取り収穫）が可能なため、播種コストを複数サイクルに分散できる点も魅力です。

### マイクログリーン（ひまわり・ブロッコリーミックス）

- **播種量**: 300g/トレイ × 2トレイ
- **初回収穫予測**: 播種から7〜10日後
- **目標販売価格**: 100g/パック 400〜600円（飲食店向け業務用EC）

マイクログリーンは播種から収穫まで**約7〜14日**という超短サイクルが最大の特徴です。週次収穫を実現できれば、バジルの月1〜2回収穫を補完し、安定した出荷スケジュールが組めます。

ひまわりとブロッコリーの混合品種は、料理の彩りとして需要が高く、高単価を維持しやすい品目です。

## 本日より開始するKPI計測

Optiensでは「**ユニットエコノミクスの証明なくして拡大なし**」を基本原則としています。データに基づかない拡大は事業の破綻につながる——過去の水耕栽培ベンチャーの失敗事例から学んだ最大の教訓です。

本日より以下の4項目を計測し、Phase 2移行の判断基準とします。

| KPI | Phase 2移行条件 |
|---|---|
| 1株あたり生産コスト | 販売価格の**40%以下** |
| 週次収穫量の変動係数（CV） | **20%以内** |
| 継続発注レストラン数 | **2軒以上** |
| IoTシステム稼働率 | **1ヶ月無停止** |

この4項目を**すべてクリア**して初めてPhase 2（本格生産）へ移行します。1項目でも未達であれば、移行を見送ります。

**計測方法:**
- 電気代: Zigbeeスマートスイッチの電力ログを自動集計
- 養液コスト: 補充量×濃縮液単価で毎回記録
- 収穫量: デジタルスケールで全量計量、Supabaseに手動入力
- 稼働率: MQTTハートビートによる自動監視

## 初日のシステム動作確認

初回設置後、午後3時時点のセンサーデータは以下の通りです。

| 計測項目 | 実測値 | 目標範囲 |
|---|---|---|
| 養液EC | 1.2 mS/cm | 1.0〜2.5 mS/cm |
| 養液pH | 6.3 | 5.5〜6.5 |
| 水温 | 19.8°C | 18〜24°C |
| 室内気温 | 22.4°C | 20〜26°C |
| 湿度 | 58% | 50〜70% |
| CO2濃度 | 612 ppm | 400〜1,000 ppm |

すべてのパラメータが目標範囲内に収まっており、システムは正常に動作しています。ECと pH は播種品目の成長ステージに合わせて随時調整していきます。

発芽期（播種後〜1週間）は養液を薄め（EC 0.8〜1.2）にして根へのストレスを抑え、成長期（2〜4週間）に標準濃度（EC 1.5〜2.0）へ引き上げる予定です。

## 今後の発信スケジュール

Phase 1期間中は以下のスケジュールでブログを更新します。

- **週次**: 栽培状況の写真+センサーデータサマリー
- **月次**: コスト集計・KPI達成状況の詳細レポート
- **収穫時**: 品質評価・レストランへのサンプル提供結果

Phase 1の目標期間は**6ヶ月（2026年4月〜9月）**。全KPIをクリアしてPhase 2へ移行できるよう、データドリブンな農業経営を実践していきます。

## Optiensが目指すもの

Phase 1はあくまでも「実証」のフェーズです。目の前のラック1台で、ユニットエコノミクスを徹底的に計測する期間です。

Optiensの本来の使命は、このシステムを全国の**就労支援事業所や地方自治体**へ展開することです。農業経験がなくても、IoTとAIの力で安定した農産物生産ができる仕組みを提供する——それがPhase 5以降に目指す姿です。

まず、ラック1台から。データが積み重なるほど、システムはより賢くなります。次回は発芽状況と最初のセンサーデータ週次レポートをお届けします。
