# 期限チェックSkill — /check-deadlines

Google Tasksから事業関連タスクの期限をチェックし、CEOにレポートする。

## 対象リスト（事業タスクのみ）
| リスト名 | リストID |
|---|---|
| AI支援事業 | cE9hVHhKNGJFTDY2bGJHWQ |
| 水耕栽培事業 | bkRfOFVIT0hpaVZpZ0dBeA |
| AI小説事業 | VHV0bERpeTJBaURZV192aQ |

※ 経営管理・マイタスク（プライベート）は対象外。

## 実行手順

1. 上記3リストのタスクを `list-tasks` で取得する（未完了のみ）
2. 各タスクの `due` フィールドを今日の日付と比較:
   - **期限超過**: `⚠ {タスクID} {タイトル} は期限 {due} を過ぎています`
   - **7日以内**: `⏰ {タスクID} {タイトル} の期限が {N}日後です`
   - **8日以上先**: レポートに含めない
3. Googleカレンダーから今後30日間のイベントを取得し、事業関連（【税務】【社保】【財務】タグ付き）のイベントがあれば表示
4. レポートをテーブル形式でCEOに報告:
   - 超過タスク → 7日以内タスク → カレンダー予定 の順
   - 問題なければ「期限チェック完了、異常なし」と1行で報告

## カレンダーイベント登録時のルール
- 定期タスクをカレンダーに新規登録する際は、必ずリマインダーを設定する
- リマインダー設定の目安:
  - 申請・届出系: 2週間前 + 3日前
  - レビュー・棚卸系: 1週間前
  - 納付・支払系: 3日前
- ※ update_event APIにリマインダーパラメータがないため、既存イベントのリマインダー変更はGoogle Calendarアプリから手動で行う
