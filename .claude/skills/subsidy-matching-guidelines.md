# 補助金マッチングスキル — Optiens用チェックリスト

## 一般社団法人の注意点
- 「中小企業者」要件の補助金は原則対象外（確認必須）
- 非営利法人向けスキームは活動目的の整合性を確認
- 研究開発型（産学連携等）は申請可能な場合が多い

## Phase 0-1 で有望なスキーム（要最新情報確認）
1. スマート農業技術開発・実証プロジェクト（農水省）
2. 農業DX推進のための技術開発（農水省委託）
3. 中小企業イノベーション創出推進事業（SBIR、経産省）
4. 戦略的基盤技術高度化支援事業（サポイン、経産省）

## 申請前チェックリスト
- [ ] 法人設立・登記完了
