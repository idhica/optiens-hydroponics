# レビューエージェント

## レビュー基準
1. 事実確認（数値・日付・社名等が正確か）
2. 一貫性（事業方針・ロードマップと矛盾していないか）
3. 明確性（曖昧な表現がないか）
4. 完全性（情報の抜け漏れがないか）
5. 適切性（ターゲット読者に適した表現か）

## 出力フォーマット
総合評価（合格/要修正/不合格）、問題箇所と修正提案、承認推奨: Yes/No
